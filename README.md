# Django Full Course - 16 Hours

A comprehensive Django learning project based on a complete Django course covering beginner to advanced topics.

## 📚 Course Overview

This project documents the journey through a complete Django Full Stack development course, covering everything from basics to advanced concepts, with real-world projects included.

## 🎯 Topics Covered

### 1. **Foundations & Setup**
- MVT (Model-View-Template) Architecture in Django
- Python & Django Installation
- Django Project Structure & Configuration

### 2. **Views & URLs (Routing)**
- Views & URLs Basics
- Creating Multiple Apps with Views and URLs
- URL Parameters (path, re_path, kwargs)

### 3. **Templates**
- Templates Folder Setup (Project-level & App-level)
- Template Basics & Variables
- Template Filters (Text, Numbers, Date)
- Template Tags & Control Flow
- Template Inheritance & Static Files
- Advanced Template Tags

### 4. **Frontend Integration**
- Bootstrap in Django
- Tailwind CSS in Django

### 5. **Database & ORM**
- ORM (Object-Relational Mapping) Concepts
- Django Models & Migrations
- Model Relationships

### 6. **Authentication & Authorization**
- User Registration System
- Admin Panel & Superuser Creation
- Model Registration in Admin
- Authentication & Permissions
- User Authentication Flows

### 7. **Forms**
- HTML Forms in Django
- Django ModelForms
- Form Validation & Processing

### 8. **Advanced Features**
- File & Image Uploads
- Pagination
- Search & Filtering
- Display Messages (Django Messages Framework)
- Class-Based Views (CBVs)
- Django Middleware
- Django Signals
- Sessions Management
- Cookies Management
- Email Sending

### 9. **Performance & Caching**
- Caching Strategies:
  - Local Memory Cache
  - File-based Cache
  - Per-view Cache
  - Template Fragment Cache
  - Database Cache
  - Redis & Memcached (Third-party)

### 10. **REST API Development**
- Django REST Framework (DRF)
- API Design & Best Practices

## 🚀 Real-World Projects Included

1. **Portfolio Website** - Showcase project with models, templates, and static files
2. **Email Sending App** - Demonstrates email functionality and forms
3. **E-commerce Application** - Complete shop with products, orders, and payment integration
